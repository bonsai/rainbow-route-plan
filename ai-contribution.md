# AI-Assisted Indoor Mobility Data Contribution System

**写真とコメントを送るだけで AI が地図データにマージする仕組み**

---

## 🎯 概要

Rainbow Route の屋内移動データは、ユーザーが写真やコメントを送るだけで AI が自動解釈し、JSON データに変換してマージする仕組みを備える。

これにより、専門知識がなくても誰でもバリアフリー情報を地図に追加できる。

---

## 🧩 データ投稿フロー

### 1. ユーザーが写真を撮る

- 段差
- スロープ
- エレベーター
- 点字ブロック
- 狭い通路
- 床材
- 混雑状況

### 2. コメントを添えて送信

> 「ここ段差あります」
> 「スロープが急です」
> 「盲導犬が歩きやすい」
> 「ロボットが詰まりやすい」

### 3. AI が自動解析

- 画像認識（段差・幅・床材・傾斜など）
- コメントの自然言語解析
- 既存データとの比較
- 差分を抽出

### 4. JSON データに変換

```json
{
  "segment_id": "B2F-ELV-03",
  "update": {
    "slope_deg": 8,
    "surface": "tile",
    "tactile_block": true,
    "wheelchair": {
      "step_free": false
    }
  }
}
```

### 5. AI がマージ提案を生成

- 既存データとの差分
- 信頼度スコア
- 人間のレビューが必要な箇所

### 6. 管理者が承認 → データ反映

---

## 🤖 AI ができること

- 段差の高さ推定
- 通路幅の推定
- 床材の分類（タイル / カーペット / グレーチング）
- 点字ブロックの検出
- 混雑度の推定
- 車椅子・ロボット通行可否の推定
- コメントの意味解析
- JSON への自動変換
- 既存データとのマージ
