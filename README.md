# 🌈 Rainbow Route

**Universal Mobility Mapping for Every Walker, Rider, Robot, and Guide.**

Rainbow Route は、ロボット・車椅子・ベビーカー・スーツケース・盲導犬ユーザー・高齢者など、あらゆる移動者のための"多様性対応ルートデータ"を整備するオープンプロジェクトです。

現代の都市は複雑で、多層構造で、ダンジョンのように入り組んでいます。しかし、現在の地図データは「歩行者の平均像」しか想定していないため、多様な移動者にとっては十分ではありません。

Rainbow Route は、Google Maps のプラグインとして動作する JSON ベースのユニバーサル移動データを構築し、すべての人・すべてのロボットが"迷わず・安全に・快適に"移動できる世界を目指します。

---

## 🚀 Mission

**「すべての移動者に、最適なルートを。」**

- 車椅子が"段差ゼロ"で移動できるルート
- ベビーカーが"押しやすい"傾斜と幅を持つルート
- スーツケースが"引っかからない"床材のルート
- 盲導犬ユーザーが"音・触覚"で安心できるルート
- ロボット配送が"衝突しない"経路
- 高齢者が"無理なく歩ける"休憩ポイント付きルート

これらを**ひとつのデータモデル**で表現するのが Rainbow Route です。

---

## 🧭 What is Rainbow Route?

Rainbow Route は以下の 3 つで構成されます。

### 1. Universal Mobility JSON

Google Maps や他の地図エンジンに読み込める多様性対応ルートデータフォーマット。

**含まれる情報例：**
- 段差・傾斜・幅
- 床材（タイル / カーペット / グレーチング）
- エレベーター・スロープ・エスカレーター
- 混雑度・音環境
- ロボット通行可否
- 盲導犬ユーザー向け触覚誘導ブロック
- ベビーカー向けの押しやすさ
- 車椅子の旋回可能スペース
- マルチレベル（地下・地上・連絡通路）構造

### 2. Micro Guide Agent（対話型ガイドエージェント）

盲者・弱視者・高齢者向けの音声・対話型ナビゲーションエージェント。

> 「右に 3 歩進むと点字ブロックがあります」
> 「次の角は T 字路です」
> 「エレベーターは左側にあります」
> 「ロボット配送ルートと交差します、注意してください」

### 3. Fleet & Walker Routing Engine

ロボット・車椅子・ベビーカーなど移動特性の異なる"多様なフリート"のためのルーティングエンジン。

- **車椅子**：段差ゼロ優先
- **ロボット**：幅・障害物・混雑度優先
- **ベビーカー**：傾斜と床材優先
- **盲導犬**：触覚誘導ブロック優先

## 🏙 Why Now?

現代都市は**複雑なダンジョン**になっている。

- 地下鉄の多層構造
- 商業施設の立体動線
- 駅ビルの連絡通路
- 地下街の迷路構造
- ロボット配送の増加
- 高齢化社会の進行

しかし、地図データはまだ**「平均的な歩行者」**しか想定していない。

Rainbow Route は、都市の複雑さを**"誰でも歩けるダンジョン"に変える**プロジェクトです。

## 🧩 Data Format (JSON Example)

```json
{
  "segment_id": "A-102",
  "level": -1,
  "width_m": 1.8,
  "slope_deg": 3,
  "surface": "tile",
  "tactile_block": true,
  "wheelchair": {
    "step_free": true,
    "turn_space_m": 1.5
  },
  "robot": {
    "allowed": true,
    "collision_risk": "low"
  },
  "baby_stroller": {
    "push_resistance": "low"
  }
}
```

## 🧪 Use Cases

- **盲者向け対話ナビ**：「あと 5 メートルで点字ブロックが途切れます」
- **車椅子ルート**：段差ゼロ・傾斜 5° 以下・幅 1.2m 以上のルートを自動生成
- **ロボット配送**：混雑度・幅・障害物を考慮した最適経路
- **ベビーカー**：押しやすい床材・緩やかなスロープを優先
- **スーツケース**：キャスターが引っかからない床材を優先
