# 🤖 Robot Fleet QR — オートスキャンによる位置同定

---

## 1. 概念

同じ QR コードを、**ロボットも読む。**

人間がスマホで QR をかざすように、
ロボットは通過時に自動スキャン（オートスキャン）する。

QR は「この場所です」という普遍的な位置標識。
読むのが人間でもロボットでも、返す情報は最適化される。

---

## 2. アーキテクチャ

```
                    同じ QR コード
                     ┌──────┐
                     │B2F-03│
                     └──────┘
                    ↙         ↘
             人間（スマホ）    ロボット（カメラ）
                   │                │
                   ▼                ▼
             サーバ ────→ それぞれに最適化された情報
```

**QR コードは人間とロボットで共用する。**
貼るのは1枚、情報の解釈はサーバ側で分岐する。

---

## 3. ロボットのオートスキャン動作

### 3.1 スキャンタイミング

- 天井・壁面・床面に設置された QR を通過時に自動検出
- 停止不要、走行しながら読む
- 一定間隔（例：10m ごと）で位置補正

### 3.2 取得情報（ロボット最適化）

```json
{
  "location_id": "B2F-03",
  "position": {
    "lat": 35.6895,
    "lon": 139.7004,
    "level": "B2F"
  },
  "robot": {
    "width_m": 1.8,
    "collision_risk": "low",
    "crowd_level": "medium",
    "surface": "tile",
    "slope_deg": 0,
    "obstacles": []
  },
  "next_qr": {
    "id": "B2F-04",
    "direction": "straight",
    "distance_m": 12
  }
}
```

### 3.3 ロボット向け音声 / データ指示

ロボットには音声ではなく、**JSON データ** で指示を送る。

```json
{
  "instruction": "proceed_straight",
  "distance_m": 12,
  "speed_limit": 1.5,
  "next_scan": {
    "qr_id": "B2F-04",
    "eta_sec": 8
  }
}
```

---

## 4. 人間とロボットの混在

同じ通路を、人間（盲導犬ユーザー）とロボットがすれ違う。

QR は両方に同じ位置情報を提供するが、
解釈は異なる。

```
人間（音声）:
「まっすぐ 12 メートル進んでください。前方にロボットがいます。注意してください。」

ロボット（JSON）:
{ "instruction": "proceed_straight", "distance_m": 12, "yield_to": "pedestrian" }
```

**QR は共通言語。**
**情報はクライアントごとに最適化される。**

---

## 5. ロボットフリート管理との統合

QR スキャンにより、フリート管理サーバは以下を把握できる。

- 各ロボットの現在位置
- 移動履歴
- 混雑しているエリア
- ロボット同士の接近（衝突回避）

### 例：フリート管理画面

```
Robot-A @ B2F-03  [12m ahead → B2F-04]
Robot-B @ B2F-07  [25m ahead → B2F-08]
Robot-C @ B2F-12  [charging]
```

---

## 6. メリット

- 人間用 QR とロボット用 QR を分ける必要がない（1枚で兼用）
- GPS 不要の屋内位置同定
- ロボットが停止せずに位置補正できる
- フリート管理と統合可能
- 既存の QR インフラをそのまま使える

---

## 7. 設置ガイドライン（ロボット向け）

| 場所 | 高さ | 向き | 備考 |
|------|------|------|------|
| 天井 | 2.5〜3.0m | 下向き | ロボット上方カメラで読み取り |
| 壁面 | 0.3〜1.5m | 水平 | 人間と共用 |
| 床面 | 0m | 上向き | 耐久性が必要、摩耗注意 |
| 通路天井 | 直線10m間隔 | 下向き | 位置補正の基準点 |

---

## 8. まとめ

> **QR は人間のためだけのものではない。**
> **ロボットもまた、QR を読んで世界を知る。**

Rainbow Route の QR インフラは、
人間とロボットが同じ空間を共有するための
**"共通の位置言語"** として機能する。
